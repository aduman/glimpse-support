Built for reading, presenting, and sharing markdown

Glimpse is designed to feel calm and native on macOS while still supporting technical documents. It focuses on rendering quality first, with editing and export tools that stay close to the document.

Readable by default

Open markdown files and view them with typography, themes, syntax highlighting, tables, and document navigation that make long files easier to scan.

Presentation-ready

Turn markdown into a slideshow-style presentation and export to formats that preserve the look of your document.

macOS-first workflow

Finder Quick Look support, keyboard shortcuts, folder browsing, and native SwiftUI design keep the experience close to the platform.
